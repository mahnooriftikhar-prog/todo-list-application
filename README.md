# todo-list-application
In this app, you can add your todo items and mark them as complete or incomplete.


This is a basic code with Flask framework.
Just flow the steps one-by-one and you are good to go

Step 1: Installation:
-> pip install Flask

Step 2:  Create following folders structure 
todo_app/
│
├── app.py
├── templates/
│   └── index.html

Step 3: Flask Application Code
-> app.py

Step 4: HTML Template
-> index.html

Step 5: Run the application - Use the following command on terminal
-> python app.py
